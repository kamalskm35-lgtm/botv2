=== WAREHOUSE TELEGRAM BOT (RAILWAY READY) ===

LANGKAH DEPLOY:

1. Upload semua file ini ke repository GitHub baru.
2. Masuk ke Railway → New Project → Deploy from GitHub.
3. Pilih repository ini.
4. Masuk ke tab Variables di Railway, tambahkan:

   BOT_TOKEN = Token dari BotFather
   SPREADSHEET_NAME = Nama Google Spreadsheet
   GOOGLE_CREDENTIALS = Copy seluruh isi file credentials.json (1:1 tanpa edit)

5. Klik Deploy.

Jika berhasil, log akan menampilkan:
Bot berjalan...

Bot siap digunakan 24 jam.
